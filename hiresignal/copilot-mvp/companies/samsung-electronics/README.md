# Samsung Electronics Copilot Data

삼성전자 회사별 취업 코파일럿 데이터 폴더다.

초기 타겟은 삼성전자 전체가 아니라 `삼성전자 DS부문`으로 잡는다. 이유는 SK하이닉스와 비교 가능한 반도체 직무/산업 맥락이 있고, 회사별 코파일럿의 강점을 가장 명확히 보여줄 수 있기 때문이다.

## 예정 대표 파일

- `copilot_kb.md`: 삼성전자 DS 최종 지식 베이스
- `copilot_kb.json`: 코파일럿 주입용 구조화 데이터
- `judgment.md`: 데이터 품질과 계속 수집 여부 판단

## 조사 우선순위

1. 공식 채용/직무 자료로 DS 직무 taxonomy를 확정한다.
2. 유튜브/인터뷰 자료로 취준생 질문과 현직자 표현을 수집한다.
3. 뉴스룸/기술자료로 HBM, 파운드리, 메모리, 패키징, AI 반도체 맥락을 보강한다.
4. SK하이닉스와 비교 가능한 역할은 공통 taxonomy로 맞추고, 삼성전자 고유 직무/조직 맥락은 별도 키로 둔다.

## 주의점

삼성전자는 DX, DS, SDC, SDS 등 범위가 넓다. 취업 코파일럿 MVP에서는 먼저 DS부문으로 범위를 제한한다.
