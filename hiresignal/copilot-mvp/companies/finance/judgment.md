# Finance Background Data Judgment

생성: 2026-06-03

## 결론

금융권 데이터는 회사별 취업 코파일럿에 필요한 배경 데이터가 맞다.

다만 금융권은 하이닉스/삼성전자/현대차처럼 하나의 회사 기술 지식을 깊게 쌓는 방식만으로는 부족하다. 은행권은 `은행권 공통 준비 구조`와 `은행별 면접/직무 차이`를 같이 관리해야 한다.

제품 관점에서는 은행별 구조를 유지하는 것이 중요하다. 회사별 채용 특색을 보여주는 것이 HireSignal의 셀링 포인트이기 때문이다.

현재 금융권은 메인 타겟이 아니므로 깊게 수집하지 않는다. 대신 IBK, 우리, 하나, KB, 신한, NH농협 폴더의 뼈대를 열어두고, 나중에 특정 은행이 타겟이 되면 해당 은행만 깊게 확장한다.

## 왜 필요한가

### 1. 금융권은 자소서, 필기, 면접이 모두 제품 기능이 된다

반도체/자동차는 직무 경험 번역이 강점이었다.

금융권은 다음이 같이 필요하다.

- 자소서: 은행 관심 증거, 고객 응대 경험, 일관된 사람상
- 필기: NCS, 경제/금융상식, 디지털금융 루틴
- 면접: PT, 세일즈, BEI, 롤플레잉, 가치관 질문
- 방어력: 자소서 문장이 꼬리질문을 버틸 수 있는지

따라서 금융권 코파일럿은 단순 첨삭보다 `준비 상태 진단 + 면접 방어력 점검`이 중요하다.

### 2. 기존 유튜브 데이터가 좋은 신호를 줬다

기존 금융권 배치에서 확인된 신호:

- 자소서는 관심, 키워드, 일관성이 중요하다.
- 사람 응대 경험이 핵심 증거다.
- 필기는 단기간 벼락치기보다 루틴 관리가 중요하다.
- 과장된 자소서는 면접에서 위험하다.
- 우리은행/하나은행처럼 은행별 면접 구조가 다르다.
- 비전공자도 가능하지만 왜 은행인지 설명해야 한다.

이 데이터는 공식 자료만으로 얻기 어려운 실제 준비 방식과 면접 체감 정보를 준다.

### 3. 은행별 차이를 만들 수 있다

- IBK: 중소기업금융, 정책금융, 디지털/IT, 공공성
- 우리: 기업금융, 개인금융, 비전공자 직무 연결
- 하나: PT, BEI, 하나FIT, 조직 적합성
- KB: UB, ICT, PB 보강 필요
- 신한: 디지털/ICT, 플랫폼/고객경험 보강 필요
- 농협: 지역/농업·공공성/디지털 전환 보강 필요

은행별 차이를 쌓으면 회사별 코파일럿의 강점이 분명해진다.

운영 원칙:

- `finance/copilot_kb.md`: 은행권 공통 준비 구조
- `finance/banks/{bank}/README.md`: 은행별 채용 특색 skeleton
- 특정 은행이 타겟이 되면 `banks/{bank}/copilot_kb.md`, `copilot_kb.json`, `judgment.md`를 추가한다.
- 금융권 공통 조언만 제공하지 않고, 최종 코파일럿은 항상 목표 은행을 먼저 물어본다.

## 현재 한계

- 공식 은행별 최신 JD가 아직 충분히 쌓이지 않았다.
- 기존 유튜브는 4건이라 우리/하나/금융권 공통 신호는 좋지만 KB/신한/NH농협은 부족하다.
- 금융권 정책/산업 자료는 그대로 쓰면 뉴스 요약이 되므로 직무 연결 규칙이 필요하다.

## 계속 진행 기준

계속 수집할 가치가 있는 자료:

- 은행별 최신 채용공고와 직무기술서
- 은행별 면접 전형 구조: PT, BEI, 세일즈, 롤플레잉
- 금융권 필기/NCS 준비 자료
- 고객 응대/영업점/기업금융 실무가 구체적인 자료
- 디지털/IT 은행 직무 자료
- 금융소비자보호, 디지털금융, CBDC/예금토큰을 직무와 연결할 수 있는 자료

우선순위를 낮출 자료:

- 금융권 일반론만 말하는 영상
- 은행 이름만 있고 실제 직무/면접 정보가 없는 자료
- 스펙 나열 중심 자료
- 조회수는 높지만 은행별 차이가 없는 범용 면접 영상

## 다음 작업

1. 지금은 금융권 확장을 멈추고 은행별 skeleton만 유지한다.
2. 금융권이 다시 타겟이 되면 IBK기업은행을 먼저 깊게 보강한다. 이유는 공식 직무소개가 비교적 구조화되어 있고, 중소기업금융이라는 차별점이 명확하다.
3. 우리은행 기업금융과 하나은행 PT/BEI는 기존 유튜브 신호를 더 구조화한다.
4. KB/신한/NH농협은 최신 채용공고와 면접 유형을 수집한다.
5. 코파일럿 테스트에서는 사용자의 고객응대 경험 1개를 입력받아 목표 은행별 자소서 문장, 꼬리질문, PT/BEI 답변으로 이어지는지 확인한다.
