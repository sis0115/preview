# Shinhan Bank Initial Notes

## 코파일럿 포지션

신한은행은 금융권 코파일럿에서 `디지털/ICT`, `고객경험`, `플랫폼 금융` 축을 우선 열어둔다.

지금은 타겟이 아니므로 깊은 수집은 하지 않는다. 구조만 유지한다.

## 예상 채용 특색

- 디지털/ICT: 모바일 채널, 뱅킹 서비스, 데이터, 보안, 플랫폼
- 일반행원: 고객 중심 서비스, 영업점 업무, 금융상품 설명
- 고객경험: 디지털과 오프라인 접점의 연결

## 자소서/면접 변환

- 앱/서비스/데이터 경험은 금융 고객 문제 해결로 번역한다.
- 고객 응대 경험은 신뢰, 정확성, 지속 관계로 번역한다.
- 디지털 지원자는 기술 스택보다 금융 서비스 안정성과 고객 영향도를 설명해야 한다.

## 보강 필요

- 최신 신한은행 채용공고
- 디지털/ICT 직무기술서
- 일반행원/디지털 면접 후기
