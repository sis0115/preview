# Woori Bank Initial Notes

## 코파일럿 포지션

우리은행은 기존 유튜브 배치에서 `기업금융 2차 면접`과 `비전공자 면접 후기` 신호가 있다.

## 자소서/면접 변환

- 기업금융 지원자는 기업 고객의 재무/산업/현금흐름/여신 리스크를 이해해야 한다.
- 비전공자는 금융자격증 유무보다 왜 은행인지, 고객 응대와 정확성 경험이 있는지를 강하게 설명해야 한다.
- 자소서 문장은 면접 꼬리질문을 버틸 수 있어야 한다.

## 면접 대비 축

- 왜 우리은행인가
- 왜 기업금융인가
- 고객/기업 상황을 어떻게 파악할 것인가
- 비전공 경험을 은행 업무와 어떻게 연결할 것인가
- 자소서 경험의 실제 역할과 성과 검증

## 보강 필요

- 공식 최신 채용공고/직무기술서 수집
- 개인금융/기업금융/IT·디지털 면접 방식 분리
