# Hana Bank Initial Notes

## 코파일럿 포지션

하나은행은 기존 유튜브 배치에서 `PT`, `BEI`, `하나FIT` 면접 신호가 있다.

## 자소서/면접 변환

- 하나은행 지원자는 일반 인성질문뿐 아니라 경험 기반 면접과 발표형 면접을 준비해야 한다.
- BEI는 경험을 과장하면 바로 약해진다. 상황, 행동, 결과, 배운 점을 구체적으로 방어할 수 있어야 한다.
- PT는 금융 이슈를 말하는 능력보다 고객/은행/리스크 관점으로 구조화하는 능력이 중요하다.

## 면접 대비 축

- PT 주제 구조화
- BEI 경험 검증
- 하나FIT 가치/조직 적합성
- 고객 응대/세일즈 상황 대응
- 디지털 금융과 고객 편의 연결

## 보강 필요

- 하나은행 공식 최신 채용공고/직무기술서 수집
- PT/BEI/하나FIT 기출 추가 수집
