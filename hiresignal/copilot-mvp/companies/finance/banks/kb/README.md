# KB Kookmin Bank Initial Notes

## 코파일럿 포지션

KB국민은행은 금융권 코파일럿에서 `UB`, `PB/자산관리`, `ICT` 축을 우선 열어둔다.

지금은 타겟이 아니므로 깊은 수집은 하지 않는다. 구조만 유지한다.

## 예상 채용 특색

- UB: 기업고객금융, 고객자산관리, 영업점 기반 범용 은행 업무
- PB/자산관리: 고객 포트폴리오, 투자성향, 장기 관계, 완전판매
- ICT: 뱅킹 시스템, 플랫폼 개발, 보안, 장애 대응

## 자소서/면접 변환

- 고객 응대 경험은 자산관리, 고객관계, 신뢰 형성으로 번역한다.
- 디지털/ICT 경험은 은행 시스템 안정성, 보안, 고객 편의로 번역한다.
- PB 지원자는 상품 지식보다 적합성, 설명의무, 장기 신뢰를 강조한다.

## 보강 필요

- 최신 KB국민은행 채용공고
- UB/PB/ICT 직무기술서
- KB 면접 유형과 기출
