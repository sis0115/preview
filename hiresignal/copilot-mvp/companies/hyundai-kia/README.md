# Hyundai Motor / Kia Copilot Data

현대자동차/기아 회사별 취업 코파일럿 데이터 폴더다.

초기 범위는 `현대자동차 중심 + 기아 PBV/EV/제조/품질 맥락 보강`으로 잡는다.

## 대표 파일

- `copilot_kb.md`: 현대차/기아 최종 지식 베이스 초안
- `copilot_kb.json`: 코파일럿 주입용 구조화 데이터
- `judgment.md`: 현재 데이터가 코파일럿 배경 데이터로 적합한지 판단

## 폴더

- `roles/`: 직무 taxonomy와 상세 직무 해석
- `sources/`: 공식 채용/뉴스/기술 출처
- `collections/youtube_batch3/`: 기존 현대차 유튜브 수집 배치
- `raw/`: 내부 분석 원본

## 현재 판단

현대차/기아는 반도체 회사들과 다른 코파일럿 구조가 필요하다.

핵심은 다음 네 가지다.

- 차량 개발과 양산 품질을 연결하는 경험 번역
- SDV/전동화/자율주행/수소/로보틱스 같은 미래 모빌리티 맥락
- 생산기술, 품질, 구매처럼 협업/조율이 중요한 직무
- 자소서 경험을 PT면접 슬라이드와 면접 질문까지 이어가는 구조

현대차/기아 코파일럿은 좋은 문장을 만드는 도구보다 `필살기 경험 2개를 직무와 PT면접까지 끌고 가게 만드는 도구`가 되어야 한다.
